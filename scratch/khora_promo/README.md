# Video Promocional de Khora (Remotion)

Este proyecto genera un video promocional de 30 segundos para Khora (1920x1080, 30fps) programado 100% en React y TypeScript.

## Requisitos
* Tener instalado **Node.js** (versión 18 o superior).

## Instrucciones de Uso

1. Abre la terminal y navega hasta la carpeta del proyecto:
   ```bash
   cd scratch/khora_promo
   ```

2. Instala las dependencias necesarias:
   ```bash
   npm install
   ```

3. Abre el reproductor de vista previa interactivo (se abrirá en tu navegador en `http://localhost:3000`):
   ```bash
   npm start
   ```

4. Para renderizar y compilar el video a un archivo **MP4** final en alta definición:
   ```bash
   npm run build
   ```
   El video final se guardará en la carpeta `out/video.mp4`.
